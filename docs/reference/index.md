<Callout type="warn">
The BitRouter API is in **early beta**. Endpoints, request/response formats, and behavior may change without notice.
</Callout>

Base URL: `https://api.bitrouter.ai`. Most endpoints live under `/v1`. Routing endpoints accept either `x-api-key: <api-key>` or `Authorization: Bearer <api-key | JWT>`. Pick an endpoint group from the sidebar.
