BitRouter publishes its documentation in machine-readable form so you can feed it directly into an agent or LLM context.

- **[`llms.txt`](https://bitrouter.ai/llms.txt)** — A curated index of the documentation in the [llms.txt](https://llmstxt.org) format.
- **[`llms-full.txt`](https://bitrouter.ai/api/docs/llms-full.txt)** — The complete documentation as a single plain-text file for full-context ingestion.
- **Per-page Markdown** — Every docs page is also served as clean Markdown via the `llms-mdx` endpoint, for fetching just the page you need.
